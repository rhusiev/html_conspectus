Конспекти Гусєва Радомира

Конспекти написані для Obsidian

Стандарт

Дата

Скорочення при записі місяця словами:

  • січ
  • лют
  • бер
  • кві
  • тра
  • чер
  • лип
  • сер
  • вер
  • жов
  • лис
  • гру

рік-місяць-день (YYYY-MM-DD)
Наприклад: 1 лютого 2012 буде 2012-02-01

Для проміжку використовується "--"
Наприклад: 1 лют 2012 - 12 тра 2013 буде 2012-02-01--2013-05-12

Якщо в другій частині проміжку не вказати частини, то вона така ж, як у першій
Наприклад: 1 лют 2012 - 12 лют 2012 буде 2012-02-01--12

Виділення

Заголовки

# Гештеґи
Щоб відділяти логічні частини

Виділення логічних блоків

Жирним
Коли треба виділити, використовувати заголовки, проте, ще занадто
Деколи може використовуватися і всередині тексту для підкреслення найважливіших речей

Виділення всередині тексту

Курсивом
Коли якась важлива частина всередині тексту
Частковим випадком це є виділення визначення
Наприклад: Наполеон I Бонапарт — французький державний діяч, полководець, перший консул Французької Республіки, імператор Французів(Вікіпедія)

Коментарі

| Підсвічена вставка з палицею
Щоб виділити особисту думку, речі, що не стосуються прямо тексту тощо
Якщо на кілька рядків, то
| Коментар
продовжується |
Наприклад: Наполеон зробив більшовицьке повстання 1905 року | Так, принаймні, сказав Микола

Стиль

Коли двокрапка у кінці виділення, ставити її після нього
Тобто "Наприклад:", а не "Наприклад:"

Теґи

Для доробки

#todo
Щоб показати, що цю частину конспекту потрібно доробити

Для питання

#question
Коли виникає питання під час запису.
Зараз часу нема шукати, але пізніше треба буде знайти відповідь

Позначення

Хвилька корелювання

~
Для показу, що якісь дві ідеї/речі корелюють, мають відношення між собою/приблизно рівні
