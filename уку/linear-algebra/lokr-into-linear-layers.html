Appendix: converting LoKr into sequence of linear layers

By the definition of Kronicker product for C∈Rup​×uq​ and D∈Rvp​×vq​:

C⊗D​=​c11​c21​⋮cup​1​​c12​c22​⋮cup​2​​……⋱…​c1uq​​c2uq​​⋮cup​uq​​​​⊗​d11​d21​⋮dvp​1​​d12​d22​⋮dvp​2​​……⋱…​d1vq​​d2vq​​⋮dvp​vq​​​​==​c11​Dc21​D⋮cup​1​D​c12​Dc22​D⋮cup​2​D​……⋱…​c1uq​​Dc2uq​​D⋮cup​uq​​D​​==​c11​d11​c11​d21​⋮c11​dvp​1​c21​d11​c21​d21​⋮c21​dvp​1​⋮cup​1​dvp​1​​c11​d12​c11​d22​⋮c11​dvp​2​c21​d12​c21​d22​⋮c21​dvp​2​⋮cup​1​dvp​2​​……⋱………⋱…⋱…​c11​d1vq​​c11​d2vq​​⋮c11​dvp​vq​​c21​d1vq​​c21​d2vq​​⋮c21​dvp​vq​​⋮cup​1​dvp​vq​​​c12​d11​c12​d21​⋮c12​dvp​1​c22​d11​c22​d21​⋮c22​dvp​1​⋮cup​2​dvp​1​​……⋱………⋱…⋱…​c12​d1vq​​c12​d2vq​​⋮c12​dvp​vq​​c22​d1vq​​c22​d2vq​​⋮c22​dvp​vq​​⋮cup​2​dvp​vq​​​c12​d11​c12​d21​⋮c12​dvp​1​c22​d11​c22​d21​⋮c22​dvp​1​⋮cup​2​dvp​1​​……⋱………⋱…⋱…​c1uq​​d11​c1uq​​d21​⋮c1uq​​dvp​1​c2uq​​d11​c2uq​​d21​⋮c2uq​​dvp​1​⋮cup​uq​​dvp​1​​……⋱………⋱…⋱…​c1uq​​d1vq​​c1uq​​d2vq​​⋮c1uq​​dvp​vq​​c2uq​​d1vq​​c2uq​​d2vq​​⋮c2uq​​dvp​vq​​⋮cup​uq​​dvp​vq​​​​​

We will just consider the case of C⊗D, and the case where D is further split into BA naturally follows from this.

As an input we get b×t×q, where b - batch size, t - number of tokens and q - embedding size

To simplify things, let us just consider a single token of a single batch:

​x=​x1​x2​⋮xq​​​​

Let us see what is the result of just multiplying x and C⊗D:

​(C⊗D)x=​c11​d11​x1​+c11​d12​x2​+⋯+c11​d1vq​​xvq​​+c12​d11​xvq​+1​+⋯+c1uq​​d1vq​​xq​c11​d21​x1​+c11​d22​x2​+⋯+c11​d2vq​​xvq​​+c12​d21​xvq​+1​+⋯+c1uq​​d2vq​​xq​⋮c11​dvp​1​x1​+c11​dvp​2​x2​+⋯+c11​dvp​vq​​xvq​​+c12​dvp​1​xvq​+1​+⋯+c1uq​​dvp​vq​​xq​c21​d11​x1​+c21​d12​x2​+⋯+c21​d1vq​​xvq​​+c22​d11​xvq​+1​+⋯+c2uq​​d1vq​​xq​⋮⋮cup​1​dvp​1​x1​+cup​1​dvp​2​x2​+⋯+cup​1​dvp​vq​​xvq​​+cup​2​dvp​1​xvq​+1​+⋯+cup​uq​​dvp​vq​​xq​​​​

We can see that there is a pattern: in each line d index increments by one every time, and every vq​ times the d index is reset. Therefore, the first vq​ elements of x are dotted with the whole row of D and multiplied by the first element of C, the next vq​ elements of x are dotted with the whole row of D and multiplied by an element from C. For example, the first vq​ elements of the first row’s sum can be rewritten: c11​⋅D[1][:] x[1..qv​], the next vq​ elements as c12​⋅D[1][:] x[vq+1​..2vq​]; the first vq​ elements of the second row as c11​D[2][:] x[1..vq​], the next vq​ elements as c12​D[2][:] x[vq+1​..2vq​], etc.

This means that we can rewrite the above expression as:

​​c11​D[1][:] x[1..vq​]+c12​D[1][:] x[vq+1​..2vq​]+⋯+c1uq​​D[1][:] x[((uq​−1)vq​+1)..uq​vq​]c11​D[2][:] x[1..vq​]+c12​D[2][:] x[vq+1​..2vq​]+⋯+c1uq​​D[2][:] x[((uq​−1)vq​+1)..uq​vq​]⋮⋮cup​1​D[vp​][:] x[1..vq​]+cup​2​D[vp​][:] x[vq+1​..2vq​]+⋯+cup​uq​​D[vp​][:] x[((uq​−1)vq​+1)..uq​vq​]​​​

We tend to take some parts of x, to be specific every vq​ element at once. We will rewrite x as a vq​×uq​ matrix:

​x′=​x1​x2​⋮xvq​​​xvq​+1​xvq​+2​⋮x2vq​​​……⋱…​x(uq​−1)vq​+1​x(uq​−1)vq​+2​⋮xuq​vq​​​​​

Now we can rewrite that matrix as:

​​c11​D[1][:] x′[:][1]+c12​D[1][:] x′[:][2]+⋯+c1uq​​D[1][:] x′[:][uq​]c11​D[2][:] x′[:][1]+c12​D[2][:] x′[:][2]+⋯+c1uq​​D[2][:] x′[:][uq​]⋮⋮cup​1​D[vp​][:] x′[:][1]+cup​2​D[vp​][:] x′[:][2]+⋯+cup​uq​​D[vp​][:] x′[:][uq​]​​=♡​

Now let us go from the other end and see, what Dx′ is:

​Dx′=​d11​d21​⋮dvp​1​​……⋱…​d1vq​​d2vq​​⋮dvp​vq​​​​​x1​x2​⋮xvq​​​xvq​+1​xvq​+2​⋮x2vq​​​……⋱…​x(uq​−1)vq​+1​x(uq​−1)vq​+2​⋮xuq​vq​​​​==​D[1][:] x′[:][1]D[2][:] x′[:][1]⋮D[vp​][:] x′[:][1]​D[1][:] x′[:][2]D[2][:] x′[:][2]⋮D[vp​][:] x′[:][2]​……⋱…​D[1][:] x′[:][uq​]D[2][:] x′[:][uq​]⋮D[vp​][:] x′[:][uq​]​​​ ​(Dx′)T=​D[1][:] x′[:][1]D[1][:] x′[:][2]⋮D[1][:] x′[:][uq​]​D[2][:] x′[:][1]D[2][:] x′[:][2]⋮D[2][:] x′[:][uq​]​……⋱…​D[vp​][:] x′[:][1]D[vp​][:] x′[:][2]⋮D[vp​][:] x′[:][uq​]​​​ ​C(Dx′)T=​c11​D[1][:] x′[:][1]+c12​D[1][:] x′[:][2]+⋯+c1uq​​D[1][:] x′[:][uq​]c21​D[1][:] x′[:][1]+c22​D[1][:] x′[:][2]+⋯+c2uq​​D[1][:] x′[:][uq​]⋮cup​1​D[1][:] x′[:][1]+cup​2​D[1][:] x′[:][2]+⋯+cup​uq​​D[1][:] x′[:][uq​]​c11​D[2][:] x′[:][1]+c12​D[2][:] x′[:][2]+⋯+c1uq​​D[2][:] x′[:][uq​]c21​D[2][:] x′[:][1]+c22​D[2][:] x′[:][2]+⋯+c2uq​​D[2][:] x′[:][uq​]⋮cup​1​D[2][:] x′[:][1]+cup​2​D[2][:] x′[:][2]+⋯+cup​uq​​D[2][:] x′[:][uq​]​……⋱…​​​

Now, if we concatinate the second row to the first, the third to them, etc… (flatten the matrix), we will get:

​​c11​D[1][:] x′[:][1]+c12​D[1][:] x′[:][2]+⋯+c1uq​​D[1][:] x′[:][uq​]c11​D[2][:] x′[:][1]+c12​D[2][:] x′[:][2]+⋯+c1uq​​D[2][:] x′[:][uq​]⋮c21​D[1][:] x′[:][1]+c22​D[1][:] x′[:][2]+⋯+c2uq​​D[1][:] x′[:][uq​]⋮⋮cup​1​D[1][:] x′[:][1]+cup​2​D[1][:] x′[:][2]+⋯+cup​uq​​D[1][:] x′[:][uq​]⋮cup​1​D[2][:] x′[:][1]+cup​2​D[2][:] x′[:][2]+⋯+cup​uq​​D[2][:] x′[:][uq​]​​​

We can see that this is the same as ♡.

If we denote turning x into x′ as reshape(x), and flattening of the matrix as flatten(M), we can write the above as:

​♡=flatten(C⋅(D⋅reshape(x))T)​
