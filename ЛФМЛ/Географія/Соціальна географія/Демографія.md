# Природний приріст
12‰ - середній щорічний природний приріст
Максимальний у [[Нігер|Нігері]]: 33‰
В [[Україна|Україні]] від'ємний

Депопуляція - зменшення кількості населення

Країни за природними приростом:
- Перший тип відтворення - просте, звужене відтворення - розвинені країни, країни перехідної економіки. Природний приріст невисокий/від'ємний
- Другий тип відтворення - розширене відтворення(3+ дітей в сім'ї в середньому)

# Статева структура:
- По світу переважають чоловіки
- У країнах першого типу більше жінок
- У країнах другого типу більше чоловіків

# Вікова структура
65+ років >12% - старіння населення

Причини старіння населення:
- Старіння знизу - мало народжується дітей
- Старіння зверху - висока тривалість життя

Середні тривалості життя:
- [[Монако]]: 90 років
- [[Чад]]: 50 років

Демографічний перехід - перехід від 2 типу до 1
