# Принцип включення-вилучення
Знайти кількість елементів в об'єднанні множин(тобто не врахувати перетини кучу разів)
Для двох: $|A \cup B| = |A| + |B| - |A \cap B|$

Принцип включення-вилучення для n множин:
$$
\begin{flalign}
    &|A_1 \cup A_2 \cup A_3 \cup ... \cup A_{n}| = &&\\
    &= \sum_{1 \le i \le n} |A_{i}| - \sum_{1 \le i < j \le n} |A_{i} \cap A_{j}| + \sum_{1 \le i < j < k \le n} |A_{i} \cap A_{j} \cap A_{k}| -\\
    &-... + (-1)^{n-1} \sum_{1 \le i_1 < i_2 < ... < i_{n-1} \le n} |A_{i_1} \cap A_{i_2} \cap ... \cap A_{i_{n-1}}| &&\\
\end{flalign}
$$


# Принцип коробок Діріхле
Якщо k+1 чи більше предметів розкладено в k коробок, то існує хоча б одна коробка, що має два або більше предметів

**Узагальнено**
Якщо N предметів розкладено в k коробках, то існує хоча б одна коробка, що має $\left\lceil\frac{N}{k}\right\rceil$ або більше предметів
