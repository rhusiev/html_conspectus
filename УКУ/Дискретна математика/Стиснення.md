# Адаптивне стиснення
Йдуть по тексту
Паралельно і будують словник, і кодують
Виходить трохи гірше, але швидше + словник однозначно робиться, і його через це не потрібно зберігати. По ходу декодування сам словник відновлюється

# lz77
Алґоритм Лемпеля-Зіва 1977 року
Більшість алґоритмів починаються на lz. Якщо там число, то скоріше за все рік, в якому створили Лемпель із Зівом
Якщо ще букви, то якось покращений їхній алґоритм

Якийсь рядок(напр. abacabacdbacacacd)
*Буфер довжини l*(краще кратна 2 кількість кілобайтів, але для нас так багато заскладно, тому нехай тут *5*)
*Код* у форматі:
<на скільки посунутися вліво в буфер, скільки символів скопіювати, який символ додати>

Проходимося по рядку(**буфер**, *символ зараз*)

| Стрічка                   | Код       | Коментар                                                                                                                      |
|---------------------------|-----------|-------------------------------------------------------------------------------------------------------------------------------|
| *a*bacabacdbacacacd       | <0, 0, a> | Буфер ще порожній                                                                                                             |
| __a__*b*acabacdbacacacd  | <0, 0, b> |                                                                                                                               |
| __ab__*a*cabacdbacacacd  | <2, 1, c> | Ми посунулися на 2 вліво в буфері, скопіювали один символ(a), ще додали c                                                     |
| __abac__*a*bacdbacacacd  | <4, 4, d> | Ми повернулися на 4 назад, скопіювали 4(тобто весь буфер(зараз його довжина ще не дійшла до 5) і ще додали d                  |
| abac__abacd__*b*acacacd | <4, 3, a> | Скопіювали bac, ще додали a                                                                                                   |
| abacabac__dbaca__*c*acd | <2, 3, d> | Повернулися на 2, скопіювали 3(тобто ca і по другому коли почали йти). Загалом cac + d. Якби не було d, <2, 3, $\varnothing$> |

**Для розкодування треба тільки код**

# lz78
Алґоритм Лемпеля-Зіва 1978 року
Тут вже не буфер, а словник
*Код* у форматі:
<якийсь номер в словнику, якийсь символ>

| Словник                           | Стрічка             | Код    | Коментар                                                   |
|-----------------------------------|---------------------|--------|------------------------------------------------------------|
| $\varnothing$                     | *a*bacabacdbacacacd | <0, a> | Далі $\varnothing$ пропускаю, але він ще є там 0 елементом |
| a                                 | a*b*acabacdbacacacd | <0, b> |                                                            |
| a, b                              | ab*a*cabacdbacacacd | <1, c> | скопіювали 1 елемент(a) і додали c                         |
| a, b, ac                          | abac*a*bacdbacacacd | <1, b> | скопіювали 1 елемент(a) і додали b                         |
| a, b, ac, ab                      | abacab*a*cdbacacacd | <3, d> | скопіювали 3 елемент(ac) і додали d                        |
| a, b, ac, ab, acd                 | abacabacd*b*acacacd | <2, a> |                                                            |
| a, b, ac, ab, acd, ba             | abacabacdba*c*acacd | <0, c> |                                                            |
| a, b, ac, ab, acd, ba, c          | abacabacdbac*a*cacd | <3, a> |                                                            |
| a, b, ac, ab, acd, ba, c, aca     | abacabacdbacaca*c*d | <7, d> |                                                            |
| a, b, ac, ab, acd, ba, c, aca, cd | abacabacdbacacacd   |        |                                                            |

# lzw
Спочатку створюємо словник з усіх літер
Потім ідемо по рядку
Якщо поточний + наступний символ є в словнику, пропускаємо цей раз. Наступного вже три символи дивимося
Якщо вже не можна далі йти, то додаємо поточний до словника, ідемо далі

Наприклад, рядок: abacabadabacdbac
Спочатку створюємо словник з усіх літер:
{0: a, 1: b, 2: c, 3: d}

**Поточний символ**, *наступний символ*:

| Стрічка                | Код | Словник  | Коментар                                                               |
|------------------------|-----|----------|------------------------------------------------------------------------|
| __a__*b*acabadabacdbac | 0   | 5: ab    |                                                                        |
| a__b__*a*cabadabacdbac | 1   | 6: ba    |                                                                        |
| ab__a__*c*abadabacdbac | 0   | 7: ac    |                                                                        |
| aba__c__*a*badabacdbac | 2   | 8: ca    |                                                                        |
| abac__a__*b*adabacdbac | -   | -        | ab уже є в словнику                                                    |
| abac__ab__*a*dabacdbac | 5   | 9: aba   | aba нема, закодовуємо тільки 5: ab                                     |
| abacab__a__*d*abacdbac | 0   | 10: ad   |                                                                        |
| abacaba__d__*a*bacdbac | 3   | 11: da   |                                                                        |
| abacabad__a__*b*acdbac | -   | -        | ab уже є в словнику                                                    |
| abacabad__ab__*a*cdbac | -   | -        | aba уже є в словнику                                                   |
| abacabad__aba__*c*dbac | 9   | 12: abac | abac нема                                                              |
| abacabadaba__c__*d*bac | 2   | 13: cd   |                                                                        |
| abacabadabac__d__*b*ac | 3   | 14: db   |                                                                        |
| abacabadabacd__b__*a*c | -   | -        | ba уже є в словнику                                                    |
| abacabadabacd__ba__*c* | 6   | 15: bac  | Це вже останнє, тому можна закодувати **не 6 і наступний рядок, а 15** |
| abacabadabacdba__c__   | 4   | -        |                                                                        |

Загальний код буде:
0 1 0 2 5 0 3 9 2 3 **6 4**
або
0 1 0 2 5 0 3 9 2 3 **15**

Але треба ще мати словник, щоб розкодувати
