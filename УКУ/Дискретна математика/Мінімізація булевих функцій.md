Можна і [[Кон'юнктивна нормальна форма|КНФ]], але різниці нема, тому працюватимемо з [[Диз'юнктивна нормальна форма|ДНФ]]

# Карти карно
Використовують для отримання всіх *тупикових* [[Диз'юнктивна нормальна форма|ДНФ]](які ще сильніше скоротити не можна)

Є якась $f = (1111\ 1100\ 0001\ 0011)$(пробіли між цифрами виключно для зручності, не несуть сенсу. Це все один рядок)

Це транслюється в:

| x    | Значення |
|------|----------|
| 0000 | 1        |
| 0001 | 1        |
| 0010 | 1        |
| 0011 | 1        |
| 0100 | 1        |
| 0101 | 1        |
| 0110 | 0        |
| 0111 | 0        |
| 1000 | 0        |
| 1001 | 0        |
| 1010 | 0        |
| 1011 | 1        |
| 1100 | 0        |
| 1101 | 0        |
| 1110 | 1        |
| 1111 | 1        |

Тепер будуємо табличку. Сусідні стовпці(і рядки) мають відрізнятися в 1 біті. Клітинки, де 0, просто порожні.

|                                | $x_1x_2$ | $x_1\overline{x_2}$ | $\overline{x_1}\overline{x_2}$ | $\overline{x_1}x_2$ |
|--------------------------------|----------|---------------------|--------------------------------|---------------------|
| $x_3x_4$                       | 1        | 1                   | 1                              |                     |
| $x_3\overline{x_4}$            | 1        |                     | 1                              |                     |
| $\overline{x_3}\overline{x_4}$ |          |                     | 1                              | 1                   |
| $\overline{x_3}x_4$            |          |                     | 1                              | 1                   |

Нам необхідно знаходити якнайбільші прямокутники $2^i \times 2^j$, що складаються з одиниць і покрити всю табличку

Прямокутники можуть накладатися, але не повинно бути такого прямокутника, кожна клітинка якого вже покрита якимось іншим(в такому випадку це *нетупикова* буде)
Це означає, що існує *не один варіант розбиття*

Наприклад(тут уже упущу всі $x_1x_2$ і тд для економії місця):

|              |       |         |       |
|--------------|-------|---------|-------|
| ==<u>1</u>== | ==1== | *1*     |       |
| <u>1</u>     |       | *1*     |       |
|              |       | _**1**_ | **1** |
|              |       | _**1**_ | **1** |

Тепер з кожного прямокутника виписуємо ті $x_i$, які сталі в усіх клітинках:
Наприклад, $x_1x_2x_3$ з підкресленого
$x_1x_2x_3 \lor x_1x_3x_4 \lor \overline{x_1}\overline{x_2} \lor \overline{x_1}\overline{x_3}$

Є й інші варіанти. Також можна циклитися(брати кінець стовпця, продовжувати з початку - тут це *курсивне*):

|              |       |       |       |
|--------------|-------|-------|-------|
| ==<u>1</u>== | ==1== | *1*   |       |
| <u>1</u>     |       | **1** |       |
|              |       | **1** | ~~1~~ |
|              |       | *1*   | ~~1~~ |


# Метод Квайна-Мак-Класкі
Узагалі Квайна та Мак-Класкі різні, але ідея одна - виконання інше

Ґрунтуються на:
- *Неповне склеювання* - $(A \land x) \lor (A \land \overline{x}) = (A \land x) \lor (A \land \overline{x}) \land A$
- *Поглинання* - $(A \land B) \lor A = A$

Одразу на прикладі
$$
\begin{flalign}
    
& f = \overbrace{\overline{a}\overline{b}c\overline{d}}^{1} \lor 
\overbrace{\overline{a}\overline{b}\overline{c}\overline{d}}^{2} \lor
\overbrace{\overline{a}bc\overline{d}}^{3} \lor
\overbrace{\overline{a}bcd}^{4} \lor
\overbrace{\overline{a}b\overline{c}d}^{5} \lor
\overbrace{abcd}^{6} \lor
\overbrace{ab\overline{c}d}^{7} \lor
\overbrace{a\overline{b}\overline{c}d}^{8} \lor
\overbrace{a\overline{b}\overline{c}\overline{d}}^{9}&&
\end{flalign}
$$

## Метод Квайна

Виписуємо:
~~1)~~ $\overline{a}\overline{b}c\overline{d}$
~~2)~~ $\overline{a}\overline{b}\overline{c}\overline{d}$
~~3)~~ $\overline{a}bc\overline{d}$
~~4)~~ $\overline{a}bcd$
~~5)~~ $\overline{a}b\overline{c}d$
~~6)~~ $abcd$
~~7)~~ $ab\overline{c}d$
~~8)~~ $a\overline{b}\overline{c}d$
~~9)~~ $a\overline{b}\overline{c}\overline{d}$

Клеїмо номери, так щоб тільки один біт відрізнявся. Записуємо те, що не змінилося:
$(1, 2) - \overline{a}\overline{b}\overline{d}$
$(3, 4) - \overline{a}bc$
$(1, 3) - \overline{a}c\overline{d}$
$\cancel{(4, 5)} - \overline{a}bd$
$\cancel{(4, 6)} - bcd$
$\cancel{(6, 7)} - abd$
$\cancel{(5, 7)} - b\overline{c}d$
$(7, 8) - a\overline{c}d$
$(2, 9) - \overline{b}\overline{c}\overline{d}$
$(8, 9) - a\overline{b}\overline{c}$

Тепер клеїмо ці:
$(4, 5, 6, 7): bd$
$(4, 6, 5, 7): bd$

Викреслюємо всі, що були використані для склеювання наступних

Тепер робимо табличку
По горизонталі - конституенти(початкові)
По вертикалі - імпліканти(склеєні)

|                                        | $\overline{a}\overline{b}c\overline{d}$ | $\overline{a}\overline{b}\overline{c}\overline{d}$ | $\overline{a}bc\overline{d}$ | $\overline{a}bcd$ | $\overline{a}b\overline{c}d$ | $abcd$ | $ab\overline{c}d$ | $a\overline{b}\overline{c}d$ | $a\overline{b}\overline{c}\overline{d}$ |
|----------------------------------------|-----------------------------------------|----------------------------------------------------|------------------------------|-------------------|------------------------------|--------|-------------------|------------------------------|-----------------------------------------|
| $\overline{a}\overline{b}\overline{d}$ | +                                       | +                                                  |                              |                   |                              |        |                   |                              |                                         |
| $\overline{a}bc$                       |                                         |                                                    | +                            | +                 |                              |        |                   |                              |                                         |
| $\overline{a}c\overline{d}$            | +                                       |                                                    | +                            |                   |                              |        |                   |                              |                                         |
| $a\overline{c}d$                       |                                         |                                                    |                              |                   |                              |        | +                 | +                            |                                         |
| $\overline{b}\overline{c}\overline{d}$ |                                         | +                                                  |                              |                   |                              |        |                   |                              | +                                       |
| $a\overline{b}\overline{c}$            |                                         |                                                    |                              |                   |                              |        |                   | +                            | +                                       |
| $bd$                                   |                                         |                                                    |                              | +                 | +                            | +      | +                 |                              |                                         |

Тепер треба взяти набір рядків, так що всі стовпці покриті були

Оскільки ми шукаємо тупикові, то варто якнайменше рядків узяти $\Rightarrow$ у кожному тупиковому буде $bd$, і можна не розглядати ті варіанти, де без $bd$. Залишилося набрати всіх інших рядків

Ми набираємо всіма можливими способами:
[[Диз'юнктивна нормальна форма|ДНФ]]:
$\overline{a}\overline{b}\overline{d} \lor \overline{a}bc \lor a\overline{c}d \lor \overline{b}\overline{c}\overline{d} \lor bd$
$\overline{a}\overline{b}\overline{d} \lor \overline{a}bc \lor a\overline{c}d \lor a\overline{b}\overline{c} \lor bd$
...

## Метод Мак-Класкі
Той самий приклад

Спочатку перетворюємо все в одиниці й нулі таким чином: $\overline{a}bc\overline{d} \to 0110$

Потім у десяткові + групуємо по кількості одиниць:

| Двійкові | Десяткові |
|----------|-----------|
| *0000*     | *0*         |
| **0010**     | **2**         |
| **1000**     | **8**         |
| ==0110==     | ==6==         |
| ==0101==     | ==5==         |
| ==1001==     | ==9==         |
| <u>0111</u>     | <u>7</u>         |
| <u>1101</u>     | <u>13</u>        |
| 1111     | 15        |

Так само клеїмо(номери - десяткові числа)(оскільки вже погруповані по кількості одиниць, то клеїмо тільки ті, що в сусідніх групах):
$(0, 2) \to 00\_0$
$(0, 8) \to \_000$
$(2, 6) \to 0\_10$
$(8, 9) \to 100\_$
$(6, 7) \to 011\_$
$(5, 7) \to 01\_1$
$(5, 13) \to \_101$
$(9, 13) \to 1\_01$
$(7, 15) \to \_111$
$(13, 15) \to 11\_1$

Тепер клеїмо ці:
$(5, 7, 13, 15) \to \_1\_1$
$(5, 13, 7, 15) \to \_1\_1$

Далі так само викреслити і табличку як у попередньому, але вже не буду повторювати
