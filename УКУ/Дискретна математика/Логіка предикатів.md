# Логіка першого порядку(Логіка предикатів)
Існують речення, які не є висловлюваннями та містять змінні(x+1=3 наприклад). Такі речення перетворюються на висловлювання, якщо надати змінним значення

x>3:
x - предмет
'>3' - предикат(або x>3 загалом)

XMORETHAN3(x), P(x)  - все з великої

Виконуваність, [[Тавтологія|загальнозначимість]], суперечність так само, як і в [[Логіка висловлювань|логіці висловлювань]]
G - виконувана<=>$\exists I, G\ в\ ній\ T$. I - модель G; I задовольняє G
G - суперечна <=> не існує інтерпретації, яка би його задовольнила
G - [[Тавтологія|Загальнозначима]] <=> кожна I задовольняє

Тут $\infty$ предметних областей => безмежно інтерпретацій => метод таблиць істинності не підійде

# Додаткові правила виведення для логіки першого порядку:
- $\lnot \forall x G(x)\equiv \exists x \lnot G(x)$
- $\lnot \exists x G(x)\ \equiv \forall x \lnot G(x)$
- $\forall x (G(x)\land H(x))\equiv \forall x G(x)\land \forall x H(x)$(для ∨ не спрацює)
- $\exists x(G(x) \lor H(x)) \equiv \exists xG(x) \lor \exists xH(x)$(для ∧ не спрацює)
Якщо ж H не від x, то можна просто винести, наприклад:
- $\forall x(G(x) \land H) \equiv \forall xG(x) \land H$
- Так само з ∨ і ∃
Ще можна переставляти однойменні квантори
- $\forall x \forall yG(x, y) \equiv \forall y \forall xG(x, y)$
- $\exists x \exists yG(x, y) \equiv \exists y \exists xG(x, y)$

- Універсальна конкретизація - якщо виконується для $\forall x P(x)$, то й для довільного $c$
- Універсальне узагальнення - якщо $\forall x P(x)$ виконується для довільного $c$, то й загалом
- Екзистенційна конкретизація - якщо $\exists x P(x)$, то існує $c$, для якого це виконується
- Екзистенційне узагальнення - якщо існує таке $c$, що виконує $P(x)$, то $\exists x P(x)$
