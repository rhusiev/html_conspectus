# Означення
*Диференціальне* рівняння $n$-го степеня - $F(x, y, y', ..., y^{(n)}) = 0$.

*Розв'язок(інтеграл)* рівняння $y(x)$ на інтервалі $I$ - якщо вона $n$ разів неперервно диференційована на $I$ і при підстановці перетворює рівняння на тотожність

*Першого порядку* - $F(x, y, y') = 0$
*Розв'язане відносно похідної* - якщо можна виразити $y' = f(x, y)$

**Диференціальна форма**:
$M(x, y)dx + N(x, y)dy = 0$

Можна малювати векторне поле, де в кожній точці є одиничний вектор $(\cos \alpha, \sin \alpha)$

*Частковий розв'язок* - такий $y = \varphi(x)$, що задовольняє $y' = f(x, y)$
*Загальний розв'язок* - сукупність $y = \varphi(x, C)$, що задовольняє
При чому:
- $y = \varphi(x, C)$ - розв'язок $\forall C$
- $\forall \varphi(x_0) = y_0: \exists ! C = C_0:$ $y = \varphi(x, C_0)$ - розв'язок

*Задача Коші* - знайти конкретний частковий розв'язок(без залежності від C)

*Диференціальне рівняння з відокремлюваними змінними* - $f_1(y)dy = f_2(x)dx$
$f_1, f_2$ - неперервні

*Однорідне диференціальне рівняння* - $P(x, y)dx + Q(x, y)dy = 0$
$P, Q$ - [[Однорідна функція|однорідні функції]] одного виміру

*Лінійне першого порядку* - $\displaystyle\frac{dy}{dx} + P(x)y = G(x)$
Якщо $G(x) = 0$ - однорідне лінійне рівняння

*Рівняння Бернуллі* - $\displaystyle\frac{dy}{dx} + P(x)y = Q(x)y^n$

## Вищих порядків
$y^{(n)} = f(x, y, y', y'', ..., y^{(n - 1)})$
або
$F(x, y, y', y'', ..., y^{(n)}) = 0$

Загальний розв'язок:
$y = y(x, C_1, C_2, ..., C_n)$

*Лінійні*:
$y^{(n)} + p_1(x)y^{(n - 1)} + ... + p_n(x)y = f(x)$
$p_i(x), f(x)$ - неперервні на $(a, b)$

*Однорідні*:
Якщо $f(x) = 0$
$y^{(n)} + p_1(x)y^{(n - 1)} + ... + p_n(x)y = 0$

Структура загального розв'язку однорідного:
$$
\begin{flalign}
    & y_{заг} = C_1 y_1(x) + C_2 y_2(x) + ... + C_n y_n(x)&&
\end{flalign}
$$

$y_{зрн} = y_{зро} + y_{чрн}$
зрн - загальний розв'язок неоднорідного
зро - загальний розв'язок однорідного
чрн - частковий розв'язок неоднорідного

Лінійне однорідне другого порядку зі сталими коефіцієнтами:
$y'' + py' + qy = 0$

# Способи розв'язання

## З відокремлюваними змінними
$f_1(y)dy = f_2(x)dx$
$\int_{}^{} f_1(y) dy = \int_{}^{} f_2(x) dx$
$F1(y) = F2(x) + C$

## Рівняння вигляду $y' = f(ax + by + c)$
Зробити заміну:
$z = ax + by + c$
$y' = f(z)$ - з відокремлюваними

## Однорідне диференціальне
$P(x, y)dx + Q(x, y)dy = 0$
Заміна: $y = z(x) \cdot x$
Зводиться до з відокремлюваними змінними

## Рівняння вигляду $(a_1x + b_1y + c_1)dx + (a_2x + b_2y + c_2)dy = 0$
Можна розв'язати:
$$
\begin{flalign}
    & \begin{cases}
        a_1x_0 + b_1y_0 + c_1 = 0 \\
        a_2x_0 + b_2y_0 + c_2 = 0
    \end{cases}&&
\end{flalign}
$$
І зробити заміну:
$\displaystyle z = \frac{y - y_0}{x - x_0}$

## Розв'язок лінійного рівняння першого порядку
$y' + P(x)y = G(x)$

### Метод варіації довільної сталої(метод Лагранжа)
1. Розв'язуємо $y' + P(x)y = 0$
$y_0 = C \cdot f(x)$ - щось вийшло
2. Уявляємо, що $C$ - $C(x)$ якась функція:
$\overset{\sim}{y} = C(x) \cdot f(x)$
$\overset{\sim}{y}' = ...$

Підставляємо це як $y$ та $y'$ в початкове рівняння
Розв'язуємо його

### Метод Бернуллі
Заміняємо $y = u(x)v(x)$. $u, v$ - деякі функції, одну з яких можна зручно собі підібрати, а іншу відносно того знайти
$y' = u'v + uv'$

$u'v + uv' + P(x)uv = G(x)$
$u'v + u(v' + P(x)v) = G(x)$
Підбираємо так, щоб $v' + P(x)v = 0$ - виходить однорідне. Знаходимо $v$.
Тоді $u'v = G(x)$
Знаходимо $u$
$y = uv$

## Рівняння Бернуллі
Робимо заміну $y^{1 - n} = z$
Розв'язується далі нормально, як лінійне

# Розв'язок вищих порядків

## Рівняння типу $y^{(n)} = f(x)$
$\displaystyle y^{(n)} = \( y^{(n - 1)} \)' = f(x)$
$\displaystyle y^{(n - 1)} = \int_{}^{} f(x) dx + C_1$
Аналогічно
$\displaystyle y^{(n - 2)} = \int_{}^{} \(\int_{}^{} f(x) dx \) dx + C_1x + C_2$
...
$\displaystyle y = \underbrace{\int \int ...}_{n} \int f(x) dx...dx + C_1 \frac{x^{n - 1}}{(n - 1)!} C_2 \frac{x^{n - 2}}{(n - 2)!} + ... C_n$

## Рівняння типу $F(x, y^{(k)}, ..., y^{(n)}) = 0$
Зробити заміну $y^{(k)} = p(x)$
Тоді $y^{(k + 1)} = p'$, $y^{(k + 2)} = p''$, ..., $y^{(n)} = p^{(n - k)}$
Підставити в рівняння
$F(x, p, p', ..., p^{(n - k)}) = 0$ - це вже вміємо

## Рівняння вигляду $F(y, y', y'', ..., y^{(n)}) = 0$
Зробити заміну $y' = p(y)$
Тоді $y'' = p'p$, $y''' = p(p''p + p'^2)$, ...

Вийде: $F(y, p, p', ..., p^{(n - 1)}) = 0$ - це вже вміємо

## Лінійне однорідне 2 порядку зі сталими коефіцієнтами
$y'' + py' + qy = 0$

Розглядають зарактеристичне рівняння:
$k^2 + pk + q = 0$

*Якщо* $D > 0$, то
Загальний розв'язок - $y = C_1e^{k_1x} + C_2e^{k_2x}$
Перевірка: $y' = C_1k_1e^{k_1x} + C_2k_2e^{k_2x}$
$y'' = C_1k_1^2e^{k_1x} + C_2k_2^2e^{k_2x}$
Підставляємо в рівняння
$C_1k_1^2e^{k_1x} + C_2k_2^2e^{k_2x} + p(C_1k_1e^{k_1x} + C_2k_2e^{k_2x}) + q(C_1e^{k_1x} + C_2e^{k_2x}) = 0$
$C_1(k_1^2 + pk_1 + q)e^{k_1x} + C_2(k_2^2 + pk_2 + q)e^{k_2x} = 0$
Оскільки $k_1$ та $k_2$ - розв'язки рівняння $k^2 + pk + q$, то обидва доданки множаться на 0 $\Rightarrow$ сума нуль, це дійсно розв'язок

*Якщо* $D = 0$, то $k_1 = k_2 = k$
Загальний розв'язок - $y = e^{kx}(C_1 + C_2x)$

*Якщо* $D < 0$, то $k_1 = \alpha + i\beta$, $k_2 = \alpha - i\beta$
$\alpha = -\frac{p}{2}$, $\beta = \frac{\sqrt{4q - p^2}}{2}$
Загальний розв'язок - $y = e^{\alpha x}(C_1\cos{\beta x} + C_2\sin{\beta x})$

## Неоднорідне

Напр $y'' - y' - 6y = x + 5$
1. Однорідне
$y'' - y' - 6y = 0$
...
$y_{зро} = C_1e^{-2x} + C_2e^{3x}$
2. Частковий розв'язок неоднорідного
$y_{чро} = Ax + B$(якщо би справа був $x^2$, то $y_{чро} = Ax^2 + Bx + C$ і тд. Деколи зі степенем тим самим не виходить, і треба брати на 1 вищий)
$y_{чро}' = A$, $y_{чро}'' = 0$
$0 - A - 6(Ax + B) = x + 5$
$-6Ax - 6B - A = x + 5$
$\displaystyle A = -\frac{1}{6}$, $\displaystyle B = -\frac{29}{36}$

3. Загальний розв'язок
$y = y_{зро} + y_{чро} = C_1e^{-2x} + C_2e^{3x} - \frac{1}{6}x - \frac{29}{36}$

Приклад 2
$y'' - y' - 6y = 2e^{5x}$
1. Однорідне
$y'' - y' - 6y = 0$
...
$y_{зро} = C_1e^{-2x} + C_2e^{3x}$
2. Частковий
$y_{чро} = Ae^{5x}$
$y_{чро}' = 5Ae^{5x}$
$y_{чро}'' = 25Ae^{5x}$
$25Ae^{5x} - 5Ae^{5x} - 6Ae^{5x} = 2e^{5x}$
$14Ae^{5x} = 2e^{5x}$
$A = \frac{1}{7}$
3. Загальний розв'язок
$y = y_{зро} + y_{чро} = C_1e^{-2x} + C_2e^{3x} + \frac{1}{7}e^{5x}$o

Приклад 3
$y'' - y' - 6y = \cos 3x$
1. Однорідне
$y'' - y' - 6y = 0$
...
$y_{зро} = C_1e^{-2x} + C_2e^{3x}$
2. Частковий
$y_{чро} = A\cos 3x + B\sin 3x$
$y_{чро}' = -3A\sin 3x + 3B\cos 3x$
$y_{чро}'' = -9A\cos 3x - 9B\sin 3x$
$-9A\cos 3x - 9B\sin 3x + 3A\sin 3x - 3B\cos 3x - 6A\cos 3x - 6B\sin 3x = \cos 3x$
$-15A - 3B = 1$, $-15B + 3A = 0$
$B = -\frac{1}{78}$, $A = -\frac{5}{78}$
$y_{чро} = -\frac{5}{78}\cos 3x - \frac{1}{78}\sin 3x$
3. Загальний розв'язок
$y = y_{зро} + y_{чро} = C_1e^{-2x} + C_2e^{3x} -\frac{5}{78}\cos 3x - \frac{1}{78}\sin 3x$

# Системи диференціальних рівнянь
$$
\begin{flalign}
    & \begin{cases}
        \displaystyle \frac{dx}{dt} = a_{11}x + a_{12}y\\
        \displaystyle \frac{dy}{dt} = a_{21}x + a_{22}y
    \end{cases}&&\\
    & a_{12}y = \frac{dx}{dt} - a_{11}x\\
    & y = \frac{1}{a_{12}} \left( \frac{dx}{dt} - a_{11}x \right)\\
    & \frac{dy}{dt} = \frac{1}{a_{12}} \(- a_{11} \frac{dx}{dt} + \frac{d^2 x}{dt^2} \)\\
    & a_{21}x + a_{22}y = \frac{1}{a_{12}} \frac{d^2 x}{dt^2} - \frac{a_{11}}{a_{12}} \frac{dx}{dt}\\
    & a_{21}x + a_{22}\( - \frac{a_{11}}{a_{12}}x + \frac{1}{a_{12}} \frac{dx}{dt} \)
    = - \frac{a_{11}}{a_{12}} \frac{dx}{dt} + \frac{1}{a_{12}} \frac{d^2x}{dt^2}
\end{flalign}
$$
