*Розбиття* $\tau$ відрізка $[a, b]$ - будь-яка скінченна система точок $x_{i}$, i = 0, 1, ..., n таку, що $a = x_0 < x_1 < ... < x_{n} = b$
$$
\begin{flalign}
    &\tau = \{x_{i}\}_{0}^{n}, |\tau|=\max_{1 \le i \le n} \Delta x_{i}&&\\
    &\Delta x_{i} = x_{i} - x_{i-1}\text{ - діаметр розбиття $\tau$}
\end{flalign}
$$

Розбиття $(\tau, \overline{\xi})$ *з вибраними точками* відрізка $[a, b]$:
$\tau$ - розбиття $[a, b]$
Кожному з $[x_{i-1}, x_{i}]$ вибрано точку $\xi_{i} \in [x_{i-1}, x_{i}]$
$\overline{\xi}=(\xi_{1}, \xi_{2}, ..., \xi_{n})$
