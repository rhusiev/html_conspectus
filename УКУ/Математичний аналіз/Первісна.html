F(x) - первісна для f(x) на ℐ, якщо F(x) диференційована ℐ і для всіх x  ∈ ℐ виконується F′(x) = f(x) (dF(x) = f(x)dx) →

Основна Теорема первісних Якщо F1(x) та F2(x) - будь-які первісні для f(x) на ℐ, то F1(x) − F2(x) = C на ℐ, C - стала Наслідок: Якщо F(x) - первісна для f(x) на ℐ, то будь-яка інша первісна Φ(x) для f(x) на ℐ має вигляд Φ(x) = F(x) + C

Невизначений інтеграл - довільна первісна функції f, ∫f(x) dx = F(x) + C

Заміна змінної в невизначеному інтегралі Нех f(x) та φ(x) визнач на Ix та It, φ(It) ⊂ Ix f(x) має первісну F(x) на Ix ⇒ ∫f(φ(t)) ⋅ φ′(t)dt = F(φ(t)) + C

Метод інтегрування частинами Нех $I \subset \R$, u, v ∈ I, ∃∫vdu ⇒ ∃∫udv, ∫udv = uv − ∫vdu 1 група: ln (x), arc-и(x), ln (φ(x)) - їх і позначаємо u(x) 2 група: (ax+b)ncos cx, (ax+b)nsin cx, (ax+b)necx: (ax+b)n = u(x) 3 група: eaxcos bx, eaxsin bx, sin ln (x), cos ln (x) - двічі інтегруємо, отримуємо початкове

Інтегрування раціональних функцій Раціональна - можна зобразити $R(x)=\frac{P_{n}(x)}{Q_{m}(x)}$ Pn(x) та Qm(x) - многочлени степенів n та m Якщо n>=m(вона неправильна), то ділити куточком Pn(x) на Qm(x), щоб вийшло правильне

Потім зображаємо Q(x) як (x−a)k(x−b)t... ⋅ (x2+αx+β)r(x2+γx+μ)s... $$ \begin{flalign} &\frac{P(x)}{Q(x)}=\underbrace{\frac{A_1}{x-a} + \frac{A_2}{(x-a)^{2}} + ... + \frac{A_{k}}{(x-a)^{k}}}_{\text{якщо є $(x-a)^{k}$ серед множників}} + \frac{B_1}{x-b} + ... \frac{B_{t}}{(x-b)^{t}} + ... + \frac{M_1x + N_1}{x^{2}+\alpha x + \beta} + \frac{M_2 x + N_2}{(x^{2}+\alpha x + \beta)^{2}} + ... + \frac{M_{r}x+N_{r}}{(x^{2}+\alpha x + \beta)^{r}} + \frac{R_1x+L_1}{x^{2}+\gamma x + \mu} + ...&& \end{flalign} $$ A1, A2, ..., B1, B2, ..., M1, N1, M2, N2, ...R1, L1, ... - якісь дійсні числа Якщо степені раціональні($x,\ x^{\frac{p_1}{q_1}},\ x^{\frac{p_2}{q_2}},...$): заміна x = tm, m = НСК{q1,q2,q3,...,qk} Якщо є щось ніби $\(\frac{ax+b}{cx+d}\)^{\frac{1}{m}}$, то заміна $\frac{ax+b}{cx+d}=t^{m}$
