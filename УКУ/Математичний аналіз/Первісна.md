F(x) - *первісна* для f(x) на $\mathcal{I}$, якщо F(x) диференційована $\mathcal{I}$ і для всіх x $\in \mathcal{I}$ виконується $F'(x) = f(x)$ ($dF(x)=f(x)dx$)

**Основна Теорема первісних**
Якщо $F_1(x)$ та $F_2(x)$ - будь-які первісні для f(x) на $\mathcal{I}$, то $F_1(x) - F_2(x) = C$  на $\mathcal{I}$, C - стала
**Наслідок**:
Якщо F(x) - первісна для f(x) на $\mathcal{I}$, то будь-яка інша первісна $\Phi(x)$ для f(x) на $\mathcal{I}$ має вигляд $\Phi(x)=F(x)+C$

*Невизначений інтеграл* - довільна первісна функції f, $\int_{{}}^{{}} {f(x)} \: d{x} {} = F(x) + C$

**Заміна змінної в невизначеному інтегралі**
Нех $f(x)$ та $\varphi(x)$ визнач на $I_x$ та $I_t$, $\varphi(I_{t)}\subset I_x$
$f(x)$ має первісну $F(x)$ на $I_x$ $\Rightarrow$ $\int_{}^{}f(\varphi(t)) \cdot  \varphi'(t)dt=F(\varphi(t)) + C$

**Метод інтегрування частинами**
Нех $I \subset \R$, $u, v \in I$, $\exists \int_{}^{}vdu$ $\Rightarrow$ $\exists \int_{}^{}udv$, $\int_{}^{}udv=uv-\int_{}^{}vdu$
*1 група*:
$\ln(x),\ \text{arc-и}(x),\ \ln(\varphi(x))$ - їх і позначаємо $u(x)$
*2 група*:
$(ax+b)^{n}\cos{cx},\ (ax+b)^{n}\sin{cx},\ (ax+b)^{n}e^{cx}$: $(ax+b)^{n}=u(x)$
*3 група*:
$e^{ax}\cos{bx},\ e^{ax}\sin{bx},\ \sin{\ln(x)},\ \cos{\ln(x)}$ - двічі інтегруємо, отримуємо початкове

**Інтегрування раціональних функцій**
Раціональна - можна зобразити $R(x)=\frac{P_{n}(x)}{Q_{m}(x)}$
$P_{n}(x)$ та $Q_{m}(x)$ - многочлени степенів $n$ та $m$
Якщо n>=m(вона неправильна), то ділити куточком $P_{n}(x)$ на $Q_{m}(x)$, щоб вийшло правильне

Потім зображаємо Q(x) як $(x-a)^{k}(x-b)^{t}...\cdot(x^{2}+\alpha x + \beta)^{r}(x^{2}+\gamma x + \mu)^{s}...$
$$
\begin{flalign}
    &\frac{P(x)}{Q(x)}=\underbrace{\frac{A_1}{x-a} + \frac{A_2}{(x-a)^{2}} + ... + \frac{A_{k}}{(x-a)^{k}}}_{\text{якщо є $(x-a)^{k}$ серед множників}} + \frac{B_1}{x-b} + ... \frac{B_{t}}{(x-b)^{t}} + ... + \frac{M_1x + N_1}{x^{2}+\alpha x + \beta} + \frac{M_2 x + N_2}{(x^{2}+\alpha x + \beta)^{2}} + ... + \frac{M_{r}x+N_{r}}{(x^{2}+\alpha x + \beta)^{r}} + \frac{R_1x+L_1}{x^{2}+\gamma x + \mu} + ...&&
\end{flalign}
$$
$A_1, A_2, ..., B_1, B_2, ..., M_1, N_1, M_2, N_2, ... R_1, L_1, ...$ - якісь дійсні числа
Якщо степені раціональні($x,\ x^{\frac{p_1}{q_1}},\ x^{\frac{p_2}{q_2}},...$): заміна $x=t^{m}$, $m = НСК \left\{q_1, q_2, q_3,...,q_{k}\right\}$
Якщо є щось ніби $\(\frac{ax+b}{cx+d}\)^{\frac{1}{m}}$, то заміна $\frac{ax+b}{cx+d}=t^{m}$
