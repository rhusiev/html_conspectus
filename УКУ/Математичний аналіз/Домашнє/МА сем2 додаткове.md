У цьому додатковому завданні я зміг написати функції для обчислення визначених інтегралів двома способами(використовував розбиття на однакові інтервали):
- За [[Означення|означенням]]
$$
\begin{flalign}
    &\int_{a}^{b} f(x) dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_{i}) \Delta x_{i}&&\\
\end{flalign}
$$
- Методом трапецій - брати середнє значення $x_{i}$ та $x_{i+1}$, по суті беручи трапеції, а не прямокутники, через що точність має бути вищою
$$
\begin{flalign}
    &\int_{a}^{b} f(x) dx \approx \(\frac{f(a) + f(b)}{2} + \sum_{i=1}^{n-1} f(x_{i})\) \cdot \frac{b-a}{n}&&\\
\end{flalign}
$$

Також я написав функцію для обчислення похибки, яка використовується для порівняння точності обох методів

Для початку імпортуємо необхідні бібліотеки: sympy для функцій та matplotlib для візуалізації

Після цього знаходимо значення інтегралу за [[Означення|означенням]] та методом трапецій
```python
def riemann_method(func, start: float, end: float, num: int) -> float:
    delta_x = (end - start) / num
    return sum(func(start + delta_x * i) for i in range(1, num + 1)) * delta_x
```

```python
def trapezoidal_method(func, start: float, end: float, num: int) -> float:
    delta_x = (end - start) / num
    sum_of_heights = sum(func(start + delta_x * i) for i in range(1, num))
    return ((func(start) + func(end)) / 2 + sum_of_heights) * (end - start) / num
```

Після цього визначаємо функції для обчислення похибок обох методів(функції `get_errors` та `compare_accuracy`)

Тепер у головній функції ми можемо порівняти точність обох методів для різних функцій значень `n`

\begin{flalign}

\end{flalign}
$$
